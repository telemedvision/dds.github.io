# Diabetes Distress Scale (DDS) Research Questionnaires

The Diabetes Distress Scale Research Questionnaires is a web-based application for administering and analyzing the DDS assessment tool, used to measure emotional and psychological distress related to diabetes management.

## Overview

This repository hosts a comprehensive research questionnaire platform designed to support diabetes distress research and clinical assessments. The application provides an interactive interface for participants to complete the DDS questionnaire and researchers to manage and analyze responses.

## Tech Stack

- **JavaScript** (62.8%) - Primary client-side logic and interactivity
- **HTML** (17.4%) - Markup and structure
- **Python** (10.7%) - Backend processing and data analysis
- **CSS** (9.1%) - Styling and responsive design

## Features

- Interactive questionnaire interface
- Data collection and validation
- Research-grade assessment tool
- Responsive web design

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/karanibranfon/dds.github.io.git
   ```

2. Install dependencies (if applicable)
   ```bash
   # For frontend dependencies
   npm install
   
   # For Python dependencies
   pip install -r requirements.txt
   ```

3. Open the application in your browser or deploy according to your setup

## Project Structure

- Frontend components built with JavaScript and HTML

- Responsive CSS styling for cross-platform compatibility
